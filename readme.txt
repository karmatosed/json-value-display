This plugin demonstrates how to use CSS values from the theme's theme.json in a plugin.

== Description ==

This plugin shows how to extract and use CSS values defined in a theme's theme.json file, specifically demonstrating with a custom admin notice using the primary color from the theme.

== Installation ==

1. Upload `json-value-display` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
